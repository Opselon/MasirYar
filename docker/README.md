# مستندات زیرساخت داکر (Docker Infrastructure)

این پوشه شامل تمام تنظیمات مربوط به `docker-compose` برای راه‌اندازی محیط توسعه محلی است.

## سرویس‌ها

### PostgreSQL
- **سرویس:** `postgres-db`
- **دسترسی:** `localhost:5433`
- **کاربر:** `admin`
- **رمز عبور:** `secret`
- **دیتابیس:** `PersonalGrowthDb`

### RabbitMQ
- **سرویس:** `rabbitmq`
- **پورت ارتباطی:** `5672`
- **داشبورد مدیریتی:** [http://localhost:15672](http://localhost:15672)

#### نحوه استفاده از داشبورد RabbitMQ
برای دسترسی به داشبورد، به آدرس [http://localhost:15672](http://localhost:15672) بروید و با اطلاعات پیش‌فرض زیر وارد شوید:
- **نام کاربری:** `guest`
- **رمز عبور:** `guest`

در این داشبورد می‌توانید موارد زیر را مشاهده و مدیریت کنید:
- **Exchanges:** مسیریاب‌هایی که پیام‌ها را از تولیدکنندگان (Producers) دریافت کرده و به صف‌های (Queues) مناسب هدایت می‌کنند.
- **Queues:** صف‌هایی که پیام‌ها را برای مصرف‌کنندگان (Consumers) نگهداری می‌کنند.
- **Connections & Channels:** اتصالات و کانال‌های فعال به سرور.

### Grafana (Observability)
- **سرویس:** `grafana`
- **داشبورد:** [http://localhost:3001](http://localhost:3001)
- **نام کاربری:** `admin`
- **رمز عبور:** `admin`

در این داشبورد می‌توانید لاگ‌های تمام سرویس‌ها را که توسط `Loki` جمع‌آوری شده‌اند، مشاهده و جستجو کنید.
